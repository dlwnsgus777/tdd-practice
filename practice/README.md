# tdd-practice

---

## 스프링 TDD 연습

스프링을 사용하여 API를 만드는 과정에 TDD 방법론을 적용해보자.

## 스펙
- Java 21
- JPA
- H2

## 기능 사항
- 친구 추천 기능을 개발한다

## 요구 사항 
- 추천 코드를 발급한다.
  - 랜덤한 숫자 7자리다.
  - 회원 정보 내에 추천 코드를 저장한다.
  - 추천 코드가 없으면 추천 코드를 생성한 후에 저장한다.
- 추천 코드를 입력하여 추천인과 피 추천인 둘 다 1000 포인트를 획득한다.
  - 추천인이 10번 추천을 받으면 포인트 적립은 되지 않는다.
